Selected work

Research projects

Transformer-based monocular UAV localization

Temporal transformer model for estimating relative 3D UAV position from cropped monocular image sequences and bounding-box features.

  • Temporal attention over image sequences
  • Uncertainty-aware prediction head
  • Trajectory and RMSE visualization

Payload detection with domain adaptation

Synthetic-to-real visual adaptation pipeline for drone payload classification using CycleGAN/CUT-style translation and CNN/ViT classifiers.

  • Real/synthetic visual domain alignment
  • Attention map visualization
  • Classifier comparison experiments

Radar-camera fusion and tracking

Analysis pipeline for synchronizing radar range estimates, GPS logs, bounding-box vision angles, and ground-truth trajectories.

  • Range, azimuth, elevation analysis
  • Radar + vision comparison plots
  • Flight log synchronization

Monte Carlo simulation platform

MATLAB/Simulink-based scenario sweeps for evaluating UAV detection, tracking robustness, weather conditions, and sensor settings.

  • Scenario batch execution
  • Tracking performance metrics
  • Weather and sensor parameter sweeps
